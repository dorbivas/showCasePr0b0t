Lets Meet

There is difficulty coordinating meetings between different people, whether it’s a meeting between teammates, meeting between a few students, catching up with friends or family, and more, due to a busy schedule that varies from person to person.

Therefore, to solve this problem we would like to develop a platform named “Lets Meet”.
The platform is meant to help the users to manage meetings of different types and centralize related information that links to those meetings such as files, chat and links to the meetings.
Our site is meant to be used by everyone.

What We Offer

Set Events

Events are the user's way to create an action that will happen in the future, or that happened in the past and wanted to be remembered.
An Event is composed of a title, date and time, location, and description.
Every event is linked to the user who created it.

Set Events

Joined Meetings

Every user can create a meeting with other users.
He will define a range of dates that he prefers for the meeting to be set, how long the meeting will take, and a list of wanted participants.

Our algorithm will return a range of dates that the meeting can be set, with all the participants involved, or it will return that there is no such date in the relevant case.

Joined Meetings

Upload Files

Users can upload and download files that they linked to an event created.
Useful examples for uploading a file are:
1. when you want to remember something that happened at the event summarize it and upload the file created.
2. can be the direction to the meeting place for meeting with friends, there are a lot of reasons to upload files especially if the event is with other people.
We support files like: word, pdf, pictures, and more

Upload Files

Set Reminders

For each event created by the user, there is an option to set a reminder to the event.
A remainder is an alert that can pop out on our site or send to the user by email.
In a joined meeting the creator can set remainders to all the participants if needed.

Set Reminders
UPCOMING FEATURES

Set Groups

Groups are for ease of use.
When a user is creating a joined meeting adding other users each time can be tidies and annoying when it is repeated multiple times with the same group of people.
Our solution is creating a group that the user will define and can be changed over time to remove the discomfort of searching for right users for the meeting each time.

Set Groups

chat

Users will be able to communicate with each other using our site within groups and within a joined meeting.

chat

External Calendar

Users will be able to bring their own calendar provider and import events from it to our site to unify the calendars into one.
At this point in time, we will be supporting google calendar.

External Calendar
